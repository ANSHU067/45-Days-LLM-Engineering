# Career Launchpad · Day 4 — The 10-Second Profile Test

Day 4 of the daily career track. Turns the platform they picked yesterday into a profile that
survives a recruiter's 10-second glance — headline, photo, bio.

> 📁 This folder currently holds only the **career talk**. The Day-4 *code* (Loops) will be added
> to this same `Day-04-Loops` folder later.

## How to present it
Self-contained `index.html`. Double-click → **F** for fullscreen → **→ / Space** next, **←** back.

## Slide flow + speaker notes
10 slides, ~28–30 min.

| # | Slide | Talk about / ask the room |
|--:|-------|---------------------------|
| 1 | Title | "We make yesterday's profile actually work." |
| 2 | 10 seconds | Recruiters scan, not read. The profile's job = survive one glance. |
| 3 | 3 things scanned | Photo · headline · first lines of bio. The rest is bonus. |
| 4 | Headline formula | Role + Niche + Proof. Ban "aspiring"/"fresher". |
| 5 | Makeovers | ⭐ Read weak vs strong aloud. Ask which they'd click. |
| 6 | 3-line bio | Walk the structure; give the worked example. |
| 7 | Photo/banner/handle | Quick do/don't. |
| 8 | Consistency | Same name/handle/photo everywhere → findable. |
| 9 | **Career Action** | Run the live 10-second pair exercise — it lands hard. |
| 10 | Close | Tease Day 5 (GitHub). |

## Today's Career Action
**Rewrite your headline** (Role + Niche + Proof) and **3-line bio** on your primary platform, add a
clear photo. Then do the 10-second partner test and fix what they couldn't recall.
